# Student Task Manager

A full-stack web application built using Python Flask that helps students manage daily tasks.

## Features
- Login System
- Add Tasks with Priority
- Set Deadline Date
- Delete Tasks
- Clean UI using CSS

## Technology Used
- Python Flask
- SQLite Database
- HTML, CSS, JavaScript

## How to Run
1. Install Flask
pip install flask

2. Run
python app.py

3. Open in browser
http://127.0.0.1:5000
